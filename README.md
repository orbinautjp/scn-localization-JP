# scn-localization-JP

Orbiter Space Flight Simulatorのシナリオファイルを日本語化するプロジェクトです。

https://github.com/orbinautjp/scn-localization-JP

## 対応バージョン

Orbiter2016で利用できます。  
Orbiter2010では利用できません。

## インストール

Scenariosフォルダをコピーする。  
Orbiterのインストールフォルダを開いて、既存のScenariosフォルダを上書き。

## 利用規約

Orbiter本体の利用規約に準じているため、営利利用はできません。  
複製・改変は自由ですが、配布する場合は元ファイルへのリンクを添付してください。  
改変した場合は変更点を明記してください。

## 注意点

### ゲーム内字幕

リプレイ等で表示される字幕は日本語に対応していません。

### 画像の表示

一部のシナリオではchmファイルの読み込みによって画像を表示していますが、日本語版ではchmファイルを利用していないため、画像が表示されません。

chmファイルは古い規格であり、今後のOrbiterでサポートされるかわからないため、このような対応になっています。

### チェックリスト

Checklistsフォルダのシナリオは翻訳していません。

「Orbiterの付属マニュアルやチェックリストを読んでも肝心なことが書いてない」という不満が、この活動を始めた原点なので、今後も触れることはないと思います。  
打ち上げやランデブーについては以下のページで解説しているので、参考にしてください。

https://orbinautjp.github.io/tutorial/
